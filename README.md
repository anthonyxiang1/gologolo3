# Logo Maker
Logo maker CRUD app

## Experience
Users can create, design, edit, and delete logos.

# Technical
## Views
- Home page
- View logo
- Edit logo
- Create logo

## Features
- Sidebar allows for designing to create/edit the logo
- Logos ordered by most recently edited

## Languages
#### Front-End
* HTML
* CSS
* JavaScript
* React-materialize
* Bootstrap

#### Back-End
* Node.js
* Express
* Mongoose
* GraphQL
* MongoDB
